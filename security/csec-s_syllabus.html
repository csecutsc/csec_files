Topics:
+ Environment 101 (Environ101)
Introductory level: Explore the linux environment and tools available. Pick
up tricks and learn how to build custom tools for automating tasks. The skills
you learn here will help ease your tasks in other categories.

+ Reverse Engineering and Exploitation 101 (RE-Exp101)
Introductory level: Understand what the computer actually does to execute
our high level code. Understand basic assembly code; the idea of registers,
instructions and the stack. Learn about common developer mistakes and how to
recognize them. Understand the basic path to exploitation.

+ Web Application and Networks 101 (Web-Net101)
Introductory level: Understand the Network stack and related protocol. Explore
the position of an attacker at any point within the stack. Packets and packet
structures. Capturing and analysing traffic. What are web-applications. The HTTP
protocol. Learn about common developer mistakes and how to recognize them. Understand
the basic path to exploitation.
